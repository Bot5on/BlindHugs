# BlindHugs

Игра, в которой два игрока пытаются найти друг друга в темноте, используя лишь звуковые волны и следы.

## Описание

BlindHugs - это игра про поиск в темноте. Два игрока находятся в одном пространстве, но не видят друг друга напрямую. Для навигации используются:
- Звуковые волны, которые расходятся от шагов игроков
- Следы, оставляемые при движении
- Эхолокация, позволяющая определять расположение препятствий

Цель игры: найти друг друга и обняться!

## Управление

### Игрок 1
- W, A, S, D - движение (вверх, влево, вниз, вправо)

### Игрок 2
- Стрелки (↑, ←, ↓, →) - движение

## Особенности игры

- Уникальная механика эхолокации
- Визуальные эффекты звуковых волн
- Следы от шагов
- Простое и интуитивно понятное управление
- Красочный пиксельный стиль

## Технические детали

- Игра разработана на движке Godot 4.4
- Использует физическую систему для определения столкновений и распространения звуковых волн
- Содержит оригинальные визуальные эффекты и анимации
